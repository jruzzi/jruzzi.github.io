[[overview]]
1 Overview
----------

[[JEETT1339]]

This chapter introduces you to Java EE enterprise application
development. Here you will review development basics, learn about the
Java EE architecture and APIs, become acquainted with important terms
and concepts, and find out how to approach Java EE application
programming, assembly, and deployment.

The following topics are addressed here:

* link:overview001.md#A1046550[Introduction to Java EE]
* link:overview002.md#GIQVH[Java EE 7 Platform Highlights]
* link:overview003.md#BNAAX[Java EE Application Model]
* link:overview004.md#BNAAY[Distributed Multitiered Applications]
* link:overview005.md#BNABO[Java EE Containers]
* link:overview006.md#BNABS[Web Services Support]
* link:overview007.md#BNABX[Java EE Application Assembly and Deployment]
* link:overview008.md#BNACJ[Java EE 7 APIs]
* link:overview009.md#GIRDR[Java EE 7 APIs in the Java Platform,Standard Edition 7]
* link:overview010.md#BNADF[GlassFish Server Tools]
